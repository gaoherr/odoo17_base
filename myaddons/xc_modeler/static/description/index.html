Funenc Xc modeler is odoo's code generator and data relation designer. It is part of our low-code system (modeler, workflow, form designer, reporter). It can collect data from odoo and generate code for odoo. It is two-way. It includes a server and a client. The client is used to generate code for odoo. The server is used to manage data. They are all written in python. it can be used in windows, linux, mac.

As developers, we always need to add some forms or change it, but we need to write code and put the files in the right place, This is a simple thing, but if you have something to gain and gain by doing this, we are too tired and too wasteful of time. And it's easy to make mistakes. Why not use tools to do it for you?

As a project manager, we need to manage the project, understand the project structure, understand the project data, and understand the relationship between the project data. So, it is also a very useful tool. This is a communication tool, we need to help other developers, we need to help other project managers, we need to help other project owners.

Since it is a separate tool, we do not need to update every odoo version. So it can be used from odoo 12.0 to 15.0. It is a tool for developers, project managers, and project owners. And has many other benefits
